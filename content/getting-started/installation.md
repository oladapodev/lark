# Installation

## npm

```bash
npm install lark-docs
```

## pnpm

```bash
pnpm add lark-docs
```

## yarn

```bash
yarn add lark-docs
```

## Manual

Clone the repo and copy the `layouts/`, `content/`, and `src/` directories into your project.
