# Welcome to Lark

**Lark** is the open-source documentation platform that looks better than Mintlify. Built for developers who give a damn about design.

## Why Lark?

Most docs platforms force you into their cloud, their themes, their pricing. Lark gives you:

- **Full control** — self-host or deploy anywhere
- **Beautiful defaults** — themes that don’t look like Bootstrap 2015
- **Fast builds** — Hugo + Tailwind = static speed
- **Customizable** — change colors, fonts, spacing in the browser

## How it works

Lark uses the same architecture as the best docs sites on the web:

1. Write content in Markdown
2. Structure it with front matter (`group`, `slug`, `toc`)
3. Build with Hugo
4. Deploy to Cloudflare Pages in seconds
